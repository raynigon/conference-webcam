# Conference Webcam

Simple App which joins the Primary Screen Video with the feed from the Primary Webcam.

## Development

Reuirements:
 - Brew is installed
 - Webcam is connected

Execute `setup-dev.sh` before opening the project in XCode.
This will install OpenCV and pkg-config.
